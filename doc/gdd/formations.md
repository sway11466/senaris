# 陣形スキル（レシピ・カタログ）

**特定のユニット配置（編成レシピ）が盤上で揃うと発動できる能動アクション**＝senaris 独自の看板機能。「配置で撃つ特殊能力」であって、ただの配置バフではない。

- **本書はレシピの正本（カタログ）**。冒険譚が増えるたびにここへ追記する（ユニット表 [units.md](units.md) と同じ「増える表」の位置づけ）。

---

## 共通ルール

- 発動は **プレイヤーの明示操作**（条件が揃っても自動では出ない）。
- **参加ユニットは行動完了**（1体は1ターンに1つの陣形スキルにのみ参加）。
- 効果は原則 **間接攻撃扱い**＝**支援は乗らない／包囲は乗る**（[combat.md](combat.md) 間接攻撃）。※バフ系は対象の攻撃/防御に準ずる。

## レシピ一覧

### ①三重詠唱（トリニティスペル）— 面殲滅
| 項目 | 内容 |
|---|---|
| レシピ | ウィザード／ウィッチ（混在可）**3体・三角形** |
| 発動 | 3体の**どれからでも**、射程5以内の対象ヘックスを指定 |
| 範囲 | 中心＋周囲6の **7ヘクス**に間接ダメージ |
| 攻撃力 | **発動者1体の実効攻撃力**（合算しない）。7ヘクスに当たる面の広さそのものが強み。損傷で威力低下 |

### ②聖なる加護の詠唱（ホーリーアリア）— 全体バフ
| 項目 | 内容 |
|---|---|
| レシピ | 占領兵（クレリック／プリースト／ビショップ混在可）**5体以上が隣接**（固まっていればOK・**形は不問**） |
| 発動 | **どのユニットからでも** |
| 効果 | 味方の**攻撃・防御 ×1.3** |
| 持続 | **次の次の自軍ターン開始まで**（自軍ターン2回＋間の敵ターン） |
| 範囲 | **マップ全体**（全味方） |

### ③神の裁き（ディバインジャッジ）— ボス狙撃
| 項目 | 内容 |
|---|---|
| レシピ | **パラディン＋占領兵2体**（クレリック／プリースト／ビショップ）・三角形 |
| 発動 | **パラディンからのみ** |
| 射程 | **10**（盤面端から端も届く超遠距離） |
| 範囲 | **単体**（屍の海を貫いて1点＝ボス狙撃） |
| 攻撃力 | **発動者（パラディン）の実効攻撃力**（①と同じ発動者ベース。単体・射程10で狙い撃つ） |

## 初出・使用

- ①②③ は [冒険譚2「アンデッドの群れ」](../campaign/tutorial2-undead-rush.md) で初確定・初教授（①=st3／②=st5／③=st7）。教える順・演出・物量との噛み合いは冒険譚側に置く。

## 実装方針（決定）

- データ化の形式：当面は `domain/formation/` にハードコード（3レシピは効果自体が個別でデータ化の旨みが薄い）。レシピが増えたら CSV/JSON 化を検討。AIの「ロジック＝コード／組合せ＝データ」とは違い、陣形は効果ごとに専用処理。
- 発動主体：当面プレイヤー専用（敵AIは陣形を使わない）。
- 三角形の判定：参加3体が相互に隣接していれば成立（hex は3マス相互隣接で三角形）。
- 段階実装：A＝フレームワーク＋①三重詠唱／B＝③神の裁き（①と同型・範囲違い）／C＝②ホーリーアリア（ターンまたぎの持続バフを新設＝最侵襲なので最後）。
