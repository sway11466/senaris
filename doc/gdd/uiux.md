# UI/UX：操作モデル

Senaris の入力・画面操作の設計。基本は本ジャンルの定番（ファイアーエムブレム／ファミコンウォーズ／ネクタリス系）の **「選択 → 移動 → コマンドメニュー」** モデルに合わせる。

対応デバイス: **マウス／トラックパッド（タッチパッド）／タッチスクリーン**。トラックパッドとタッチはジェスチャ（タップ・ピンチ・ドラッグ）を共通とし、差分のみ末尾に記す。

---

## 基本モデル：選択 → 移動 → コマンド

1. 自軍ユニットを**選択**（移動範囲=緑／攻撃範囲=赤を表示）。
2. **移動先を指定 → 移動**すると、その場で**ユニットコマンドメニューが自動で開く**。
3. メニューから行動を選ぶ（攻撃／陣形スキル／占領／待機 など）。

「動かさずに行動したい」場合は、**自分のマスを選べば**（移動距離0で）メニューが開く。

---

## デバイス別 操作表

| 操作 | マウス | トラックパッド | タッチスクリーン |
|---|---|---|---|
| 選択／移動／攻撃／決定 | 左クリック | タップ（1本指） | タップ |
| キャンセル／戻る／メニューを閉じる | 右クリック ／ Esc | 2本指タップ | 長押し（or 画面の戻る） |
| カメラ移動（パン） | **空き地**を左ドラッグ | 2本指スクロール | **空き地**を1本指ドラッグ |
| ズーム | ホイール | ピンチ | ピンチ |
| 全体表示 | F キー | F キー | 画面ボタン |
| ターン終了 | ターン終了ボタン ／ Enter | ボタン ／ Enter | 画面ボタン |
| システムメニュー | 歯車アイコン ／ Esc | 歯車 ／ Esc | 歯車 |
| ユニットコマンド | 移動後に自動で開く | 同左 | 同左 |
| 出撃（自軍拠点） | 拠点を選択 → メニュー | 同左 | 同左 |

---

## ユニットコマンドメニュー

移動後（または自マス選択後）に**自動で開く**。状況に応じて項目を出し分ける:

- **攻撃** … 対象を指定して攻撃（射程内に敵がいるとき）。
- **陣形スキル** … 配置で発動する特殊行動を起動（条件を満たすとき）。
- **占領** … 占領可ユニットが拠点上にいるとき。
- **待機** … その場で行動終了（移動だけ／動かずに終わる、いずれもここ）。
- **キャンセル** … 移動を取り消して選択し直す（右クリック／長押し／Esc でも同じ）。

> 出撃も同じ発想で、拠点を選択 → メニューに **「出撃」** を置き、控え（garrison）を隣接マスへ1体ずつ出す。右クリック専用にはしない（タッチ・トラックパッドで困らないため）。

---

## ターン終了・システムメニュー

- **ターン終了** … 画面に**常時表示のボタン**（毎ターン使うのでメニューに埋めない）。キーボードは Enter でも可。
- **システムメニュー** … 歯車アイコン（または Esc）。セーブ・ロード・設定・投了など、頻度の低い操作をまとめる。

---

## カメラ

- **パン** … **空き地からのドラッグ**（マウス左ドラッグ／タッチ1本指ドラッグ／トラックパッド2本指スクロール）。**ユニットの上からのドラッグはパンしない**（＝クリック／タップ扱い。ドラッグでユニットを動かす機能は持たない）。
- **ズーム** … マウス=ホイール、トラックパッド・タッチ=ピンチ。**カーソル位置（タッチは2本指の中心）を基点**に拡大縮小する。
- **全体表示** … 盤全体が収まる位置・倍率へ戻す（F キー／タッチは画面ボタン）。**ステージ読み込み時は自動フィット**。
- 盤は HUD（タイトル・情報パネル等）を避けた領域に収める。

---

## クリック／ドラッグの判別

押してから一定距離（数px）動いたら **ドラッグ＝パン**、動かさずに離したら **クリック／タップ＝選択・決定** とみなす（しきい値方式）。タッチの「タップ vs スワイプ」も同じ判別。

---

## デバイス差分（トラックパッド vs タッチ）

ジェスチャは共通だが、実機の前提が3点異なる:

- **トラックパッド**: カーソルがある → **ホバー強調が効く**。**2本指タップ＝右クリック相当**。キーボード併用可。
- **タッチ**: カーソルなし → ホバー無し（フィードバックはタップ後）。右クリック無し → **長押しで代替**。キーボード無し → **ターン終了・全体表示は画面ボタンが必須**。

## 実装状況（2026-06-29 時点）

段階実装の方針:

- **フェーズ1（実装済み）— カメラ操作** （[../../presentation/board/hex_board.gd](../../presentation/board/hex_board.gd)）
  - パン＝**2本指スクロール**、または**空き地から左ドラッグ**（左ドラッグはクリック／ドラッグをしきい値 `DRAG_THRESHOLD` で判別。ユニット上からのドラッグはパンしない）。
  - ズーム＝**ピンチ**（カーソル基点）。全体表示＝**F**。読み込み時に自動フィット。
  - **Windows トラックパッドのイベント実測（Godot 4.7）**: ピンチ＝`Ctrl＋ホイール`、2本指スクロール＝`修飾なしホイール`（縦=WHEEL_UP/DOWN・横=WHEEL_LEFT/RIGHT）で届く。**Ctrl の有無で判別**し、修飾なし＝パン／Ctrl付き＝ズームに割り当てている（ブラウザの zoom と同じ流儀）。
  - 既知の制約: 素のマウスホイールと2本指スクロールは同一イベントのため、マウス使用時はホイール＝パンになる（ズームは Ctrl＋ホイール／ピンチ）。マウス運用を始めたら割り当ての切替を検討。
- **フェーズ2（実装済み）— ユニットコマンドメニュー**
  - フロー: 自軍ユニットを選択（移動範囲=緑）→ **自マス or 到達マスをクリックで `PopupMenu` を開く**（移動は**未確定**＝プレビュー表示）→ `攻撃`／`待機`／`キャンセル`。
    - `攻撃`: 移動を確定 → 攻撃対象（赤リング）をクリックで攻撃。射程内に敵がいない移動先では項目を**無効化**。
    - `待機`: 移動を確定し、その駒の行動を終了（移動だけ／動かず終わる）。
      - 占領可ユニットが**敵/中立の拠点へ進む移動先**では、この項目のラベルが **`占領`** になる（動作は待機と同じ＝移動確定で domain が自動占領。誤操作防止のラベル変更）。
    - `キャンセル`: 移動操作そのものを取りやめ＝**選択解除**（駒はその場・未行動のまま残る。**undo不要**＝そもそも動かしていない）。移動先を選び直すだけなら別マスを直接クリックすればよいので、`キャンセル`は丸ごと取りやめに割り当てる。
  - **Esc／メニュー外クリック**＝選択状態へ戻る（＝移動先の選び直し）。Esc はさらに 選択→出撃モード と1段ずつ戻る。
  - ドメイン追加: `attack_targets_from`（仮移動の攻撃可否）／`set_done`＋`is_done`（待機）。テスト: `tests/unit/test_command_actions.gd`。
  - 当面メニューに出さない: **陣形スキル**（ドメイン未実装）。**占領**は専用項目ではなく、上記の`待機`→`占領`ラベル化で表現。
- **フェーズ2b（実装済み）— 出撃の拠点メニュー化＋右クリック=キャンセル**
  - 自軍の出撃可能な拠点をクリック → `PopupMenu`（`出撃`／`キャンセル`）→ `出撃` で候補マスをハイライト → クリックで出撃。拠点上に駒がいれば**駒の選択を優先**。
  - **右クリック＝キャンセル・戻る**（Esc と同じ＝メニュー→選択→出撃モードを1段ずつ解除）。出撃モード入りの役割は廃止。
  - 長押し（タッチのキャンセル）はフェーズ4で対応。
- **フェーズ3（実装済み）— ターン終了ボタン＋システムメニュー**（[../../presentation/ui/hud.gd](../../presentation/ui/hud.gd)）
  - 永続HUD（`presentation/ui/hud.gd`）を main が常設。盤の入力を邪魔しないよう自身は素通し（ボタンだけ拾う）。
  - **ターン終了ボタン**（右下・常時表示）＝ Enter と同機能。**自手番のみ有効**（AI手番・決着後は無効化）。
  - **⚙ システムメニュー**（歯車ボタン、または盤の**最上位 Esc**＝戻る対象が無い時に開く）: `リスタート`（現ステージ再読込・動作）／`セーブ`・`ロード`・`設定`（**未実装＝無効表示**）／`閉じる`。
  - HUD は進行を持たず、押下をシグナル（`end_turn_requested`／`restart_requested`）で main（`MatchController` 所有者）へ委ねる。
- **フェーズ4（未実装）**: タッチ対応（`InputEventScreenTouch`／`ScreenDrag`／長押し）。
- **現在の盤入力**: 左クリック=選択→（移動先クリックで）コマンドメニュー／自軍拠点クリックで拠点メニュー、右クリック=キャンセル・戻る、Esc=戻る（最上位でシステムメニュー）、Enter or ボタン=手番終了、歯車=システムメニュー。
