# 冒険譚：チュートリアル２「アンデッド・ラッシュ」（仮題）

senaris の**看板機能＝陣形**を正面から見せる第二の導入冒険譚。**首都から離れた主要都市の近郊で、墓地から屍が起き出す。騎士団・教会・魔法協会の混成調査隊が、湧き出すアンデッドを押し返し、元凶のネクロマンサーを討つ**という筋を通しながら、冒険譚1 で学んだ基礎の上に **魔法兵（長射程）・輸送・陣形** を1つずつ重ねていく。

> 冒険譚1 ＝ [tutorial1-goblin-raid.md](tutorial1-goblin-raid.md)（移動・地形・包囲・支援・間接・占領の基礎・釣り）。本冒険譚はそれらを**習得済み**として始まる。

---

## 設計方針

- **冒険譚1 の続き物ではなく、別の街の別の戦い**（主役も別キャスト）。基礎は履修済み前提で、新要素＝**陣形・輸送・魔法兵**に集中する。
- **陣形が背骨**。senaris 独自の看板機能（[../gdd/combat.md](../gdd/combat.md) 陣形）を、攻撃・支援・狙撃の3タイプで段階的に教える。
- **敵＝アンデッド（物量）**。倒しても湧く屍の群れを、陣形（面殲滅・継戦バフ）と輸送（迅速展開・回復ローテ）で捌くのが通しテーマ。「数的不利を覆す」（[../concepts.md](../concepts.md)）の陣形版。
- **ボス＝ネクロマンサー**。魔王はさらに奥にいる引きとして温存（次以降の冒険譚へ）。
- **難易度カーブ**: **st1〜5 は寛容**（テキトーでも勝てる＝教える回）／**st6・7 は腕試し**（ちょっと考えないと勝てない＝数的不利・手数制限気味で、習った道具を正しく使わせる）。冒険譚1（前半は格上で快勝・終盤で手応え）と同じ思想。
- **戦力供給モデル**は原則「独立」（各話完結）。[../gdd/map.md](../gdd/map.md) 戦力供給モデル参照。

## 舞台・設定

- **首都ではない主要都市**（首都から少し離れた地方都市）。最強がいる場所ではないが、街としては相応の戦力を持ち、**パラディンが"その都市の切り札"**として終盤に登場する。
- **聖職と魔法の国**：神殿（クレリック／プリースト／ビショップ）と魔法ギルド（ウィザード／ウィッチ／ソーサラー）が厚い土地柄。これが本冒険譚に**術者・聖職が大量に出る理由**を世界観で裏打ちし、陣形を主役に張る必然性になる。
- **発端**：町外れの墓地でスケルトンが目撃される → 混成の調査隊が派遣される。

> 固有名（都市名・国名・人物名）は後付けでよい（[authoring.md](authoring.md) 方針）。現状は役職・機関ベースで進める。

## 会話の登場人物（機関ベース・見習いなし）

調査隊は3つの機関の**専門家**で構成。冒険譚1 の「素人に教える」型ではなく、**専門家が自分の手の内を語る／未知の敵（屍）に気づく**形でチュートリアルのヒントを会話へ織り込む（例：魔女が「協会の合体魔法を使う。私とウィザードで詠唱を合わせれば群れごと薙げる」と言えば、それがそのまま操作ヒントになる）。

会話パートの作成方針・書式は [authoring.md](authoring.md)（共通）を参照。話者:

- **騎士**（騎士団。落ち着いた指揮・取りまとめ役。盤上では**ナイト**＝守りの壁）
- **司祭**（教会。状況報告・加護・回復。盤上では**プリースト／ビショップ**）
- **魔導師**＋**魔女**（魔法協会。後衛火力・合体魔法。盤上では**ウィザード＋ウィッチ**）
- **聖騎士**（都市の切り札。st7 で飛空艇により合流。盤上では**パラディン**）

## 陣形レシピ（本冒険譚で確定）

senaris 独自の陣形（[../gdd/combat.md](../gdd/combat.md) 陣形）を、この冒険譚で初めて具体レシピとして確定する。**3種をクラスと効果で割る**：①攻撃（術者）／②支援（聖職）／③狙撃（パラディン）。

共通ルール：
- **参加ユニットは行動完了**（陣形の基本所作。1体は1ターンに1陣形のみ）。
- **間接攻撃扱い**＝**支援は乗らない／包囲は乗る**（[../gdd/combat.md](../gdd/combat.md) 間接攻撃）。

### ①三重詠唱（トリニティスペル）— 面殲滅

| 項目 | 内容 |
|---|---|
| レシピ | ウィザード／ウィッチ（混在可）**3体・三角形** |
| 発動 | 3体の**どれからでも**、射程5以内の対象ヘックスを指定 |
| 範囲 | 中心＋周囲6の **7ヘクス**に間接ダメージ |
| 攻撃力 | 参加3体の**実効攻撃力の合算**（傷んでいなければほぼ相手を殲滅。損傷すると威力低下） |
| 狙い | 押し寄せる屍の波を一掃する快感。陣形が物量への解だと体感させる |

### ②聖なる加護の詠唱（ホーリーアリア）— 全面バフ

| 項目 | 内容 |
|---|---|
| レシピ | 占領兵（クレリック／プリースト／ビショップ混在可）**5体以上が隣接**（固まっていればOK・**形は不問**） |
| 発動 | **どのユニットからでも** |
| 効果 | 味方の**攻撃・防御 ×1.3** |
| 持続 | **次の次の自軍ターン開始まで**（自軍ターン2回＋間の敵ターンをカバー） |
| 範囲 | **マップ全体**（全味方） |
| 狙い | ①が一瞬の面攻撃なら、②は1ラウンド超持続の全体強化。敵が強くなる中盤を底上げで支える。**①③＝3体・三角に対し、②だけ「5体以上を密集」**＝重い代償（聖職5体を行動完了＋固まって囲まれやすい）で全体×1.3と釣り合う |

### ③神の裁き（ディバインジャッジ）— ボス狙撃

| 項目 | 内容 |
|---|---|
| レシピ | **パラディン＋占領兵2体**（クレリック／プリースト／ビショップ）・三角形 |
| 発動 | **パラディンからのみ** |
| 射程 | **10**（盤面端から端も届く超遠距離） |
| 範囲 | **単体**（屍の海を貫いて1点を撃つ＝ボス狙撃） |
| 攻撃力 | 参加3体の**実効攻撃力の合算**（聖職は弱いので**実質パラディン主体**） |
| 狙い | 屍に阻まれてネクロマンサーへ近づけない → 後方で隊形を組み、湧く屍を凌ぎながら神の裁きを撃ち込んで葬る |

## ステージ骨子（全7話）

| # | 教える要素 | シナリオ | 勝敗条件 |
|---|---|---|---|
| 1 | **魔法ユニット**（長射程の間接） | 墓地でスケルトン遭遇。前衛（ナイト＋ファイター）＋後衛魔法の黄金パターン | 敵殲滅 |
| 2 | **輸送①** | 広がる戦線、群れへ馬車でユニットを迅速展開 | 敵殲滅 |
| 3 | **陣形①＝三重詠唱（＋バリケード）** | ゾンビ／スケルトンの大群をバリケードで止め、合体魔法で一掃 | 敵殲滅 |
| 4 | **輸送②（＋墓地の占領）** | 離れた墓地を殲滅。強キャラ混じりで歩兵を輸送補充、聖職者で墓地を占領して湧きを止める | 敵殲滅 |
| 5 | **陣形②＝聖なる加護の詠唱** | 敵が強化、聖職5体の加護で押し返す | 敵殲滅 |
| 6 | **総合①**（腕試し） | st2〜5の要素全部入り。難度を上げる | 敵殲滅 |
| 7 | **総合②＝陣形③＝神の裁き**（ボス） | 飛空艇でパラディン空輸 → 屍を凌ぎ神の裁きでネクロマンサー討伐 | **ボス撃破** |

throughline：冒険譚1 同様、各話の戦闘後で**次話の教える要素を予告**して1本につなぐ。

### 各ステージの要点メモ

- **st1**：敵＝**ゾンビ＋スケルトン**。ナイト（壁）＋ファイター（手数）で受け、後方のウィザード／ウィッチ（射程4〜6）で安全に削る。新しさは**長射程**。少数・易しい。
- **st2**：敵に**ゴースト**が加わる。歩兵・魔法では倒せない（対空不可）→ 鈍足のプリーストを**馬車で前線へ運んで**退治。輸送①＋対空を体感。
- **st3**：ゾンビ／スケルトンの**大群**。**バリケードで止めて**ウィザード／ウィッチの三重詠唱で一掃。
- **st4**：少し離れた**墓地を殲滅**。ちょいちょい強キャラが混じる→歩兵を**輸送で補充**。聖職者で**墓地を占領＝湧き止め**。ゴーストも散発。
- **st5**：強キャラが増えてナイト＋ファイターでは辛い → 散らばった**聖職者を集めてホーリーアリア**で全体強化。固める＝囲まれるの取捨。
- **st6**：st4 を難しくした腕試し。陣形①要員も要る。st2〜5 の道具を正しく使わないと勝てない難度。
- **st7**：大量の墓地から**大量に湧く**＋ネクロマンサー。**飛空艇でパラディン到着→神の裁き**。到着までは**陣形②で耐える**。陣形＝看板の集大成。ボスは1〜2発耐える耐久にして「陣形を守り切れるか」を腕試しに。
