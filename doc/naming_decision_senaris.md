## 2. 由来・命名意図 【指針】

- **語源**: ラテン語 *senarius* / *seni*(「6つ一組の」「6つずつの」)。
- **隠し意味**: 包囲が成立する **6つの隣接ヘクス＝6** を、響きの裏に仕込んだ名前。
- **設計オマージュ**: 末尾 `-is` は **Nectaris** へのオマージュ。
  ネクタリスが「Mare Nectaris＝神酒の海＝月(舞台)」を隠していたのと同じ構造で、
  「響きはかっこいいが、実は意味が仕込まれている」造語手法を踏襲。
- **狙い**: コア機構(包囲)を、説明くさくならない形で固有名に織り込む。

---

## 3. 選定理由 【指針】

検索ヒットのしやすさ(独自性・サービス名としての発見可能性)を主軸に5候補を実調査し、Senaris を採用。

- **ゲーム/ソフト/大手ブランドの競合がゼロ**。
- 既存の被りは「スペイン／ガリシア系の姓 Señaris(ティルダ付き)」と、
  ルーマニアの小規模ゲーマーコミュニティ(Senaris Network / senaris.net)、
  GitHub 個人ユーザー程度。いずれも非商用・小規模でブランド上の障害にならない。
- 完全な独自綴りのため、ウィッシュリストが少し付くだけで検索1ページ目を独占しやすい。
- 検索強さ・独自性・意味性の3点を同時に満たした唯一の候補。
