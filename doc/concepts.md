# Senaris — コンセプト

Nectaris 系のヘックス制ターン戦術 SLG を、ファンタジーでオリジナル化した趣味プロジェクト。
生産で拡大するのではなく、**与えられた戦力を盤面で噛み合わせて勝つ**ゲーム。

- ジャンル: ヘックス制ターン戦術 SLG（生産なし・戦力固定）
- 舞台: 剣と魔法のファンタジー
- 技術 / 配布: Godot 4（2D）／まず Steam（PC）
- タイトル: *senarius*（6つ一組）＝包囲の隣接6ヘックス。末尾 `-is` は Nectaris へのオマージュ。

## 面白さの核
- 生産なし — 補充がないので 1体1体が貴重。損耗が戦略になる。
- 包囲 — 敵を囲んで弱らせる、受動・連続の下地（Nectaris の ZOC を呼び替え）。
- 陣形スキル（オリジナル） — 配置が揃うと発動する能動・離散の決め技。
- 包囲と陣形スキルは「固める ↔ 形を作る」で引っ張り合い、それが読み合いの深さになる。

## 方針
- IP: メカニクス（ルール・数値）は流用、表現（名前・絵・世界観・マップ）は総入れ替え。ネクタリスの権利はコナミ側にあるため **デッドコピーに見えない作りを徹底** する。バランス数値はルール＝システム側の領域で著作権の保護対象ではないため、初期ベースラインとして採用してよい（→ 自作ロスター確定後に再チューニング）。
- 作り方: コア（包囲）を最小プレイアブルで検証 → 手応えが出たら肉付け。実装着手はオーナー許可後。
- 陣営: 味方＝人・種族・職能／敵＝魔物。同じ性能に陣営別のラベル（名前・見た目）を貼る＋例外（片陣営専用の強ユニット少数）。
