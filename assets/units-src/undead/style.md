# アンデッド陣営（undead）ユニットの見た目ルール

アンデッド陣営のスキンを生成するときの正本。全陣営共通のルール（絵柄・陣営カラー＝[doc/art/direction.md](../../../../doc/art/direction.md)／飛行の浮遊必須・制作スペック・STYLE文と敵陣営の共通方針＝[doc/art/units.md](../../../../doc/art/units.md)）を先に読む。冒険譚の設定は [doc/campaign/tutorial2-undead-rush.md](../../../../doc/campaign/tutorial2-undead-rush.md)。

- 基調色＝骨の白〜灰（漂白された骨）＋腐肉の緑＋墓土の茶。金属は錆と黒ずみ、布はボロ。冷たく彩度を落とす（muted 維持）。味方の鋼青、ゴブリンのオリーブ緑と並べて別物に見えること。
- 「動いている死体」のサイン＝眼窩／瞳の**冷たい青緑の燐光**。全個体に小さく入れて陣営を一目で束ねる。
- 装備は生前のものが朽ちた状態（錆びた鎧・ボロ布・欠けた刃）。ゴブリンの「拾い物・粗製」とは別で、**元は正規の装備が腐った**という質感にする。
- 陣営内アンカー＝スケルトン（雑兵）。他の個体は SUBJECT に「same bleached bone / same cold blue-green eye-glow as the skeleton」等を書いて展開する。
- 機能サイン：遠隔役＝**骨の弓＋呪いの矢**（この陣営の遠隔は弓で揃える）。魔法＝**ネクロティックな青緑の燐光**。飛行＝浮遊＋足元に影（全陣営共通ルール）。
- 発光は控えめに封じ込める（強い光線・オーラ・パーティクルは出さない）。combat 立ち絵は切り抜きが荒れるため特に厳守＝[units.md](../../../../doc/art/units.md) §3.3 `cast` の注記と同じ運用。
- 各ユニットの生成 SUBJECT は同フォルダの `{skin_id}/{skin_id}_prompt.txt`。

## 個体の見分け（7種）

性能（type）は味方からの借用だが、見た目には持ち込まない（聖職記号・魔術帽などは使わない）。

| ユニット | 借用type | 見分けの特徴 |
|---|---|---|
| スケルトン | priest | 雑兵・アンカー。漂白された骨、眼窩に冷たい燐光、朽ちた革帯と錆びた片手剣。脆いが攻撃は鋭い |
| ゾンビ | bishop | 壁役。緑がかった腐肉の膨れた体、ボロ布、引きずる前傾歩き。武器を持たず腕を突き出す（硬いが攻撃弱い） |
| スケルトンアーチャー | slinger | 機敏な遠隔。細身の骨格、骨の短弓＋呪いの矢の矢筒。刃物は持たない |
| グール | fighter | 捕食者。痩せこけた灰色の屍鬼、長い爪と裂けた口、前傾の獣じみた姿勢。武器を持たず爪で襲う |
| デュラハン | warlord | 最強格。首の無い黒錆の重装騎士（頭部が無く首元に燐光）、大剣。威圧的で最も重いシルエット |
| ゴースト | pixie | 飛行の霊。半透明で裾が霞に溶ける浮遊体、足元に影＋宙に浮く（飛行の共通ルール） |
| ネクロマンサー | sniper | ボス。深いフードの朽ちたローブ、骸骨の手、骨と宝珠の杖、青緑の燐光。屍の壁の奥から呪詛を撃つ |
