# 味方（player）ユニットの見た目ルール

味方陣営のスキンを生成するときの正本。全陣営共通のルール（絵柄・陣営カラー＝[doc/art/direction.md](../../../../doc/art/direction.md)／飛行の浮遊必須・制作スペック・STYLE文＝[doc/art/units.md](../../../../doc/art/units.md)）を先に読む。

- 基調色＝青〜鋼色（スティールブルー）＋銀の鎧・白・金の差し色。端正な人間体型（direction §2）。
- 各ユニットの生成 SUBJECT は同フォルダの `{skin_id}/{skin_id}_prompt.txt`。

---

## 1. 役割（兵種）デザイン言語（shape = role）

陣営に関係なく、兵種ごとに「シルエット＋必ず持つ記号」を固定する。色が抜けても役割が読めるように。

| 兵種 | シルエット | 必ず付ける記号 |
|---|---|---|
| 歩兵 | 中肉・地に足 | 剣（小＝新兵／標準／大剣で段階）※大盾は壁役ナイトの専用記号 |
| 占領兵（聖職） | ローブ・フード | 十字モチーフ（占領役）※杖は魔法兵と被るので不可。手持ち十字／十字ロッド／十字クロジャー等で段階化 |
| 弓兵 | 細身 | 弓＋矢筒 |
| 魔法兵 | ローブ＋とんがり帽 | 杖／オーブ＋発光 |
| 斥候 | 小柄・前傾フード | 短剣・軽装 |
| 飛行兵 | 翼・浮遊 | 翼＋足元に影で浮遊表現（全陣営共通の必須ルール＝units §1） |
| 精鋭 | 一回り大きく豪華 | 重厚な装備＋オーラ（強そう） |
| 輸送 | 非戦闘・乗り物 | 荷車／船（人型でない） |
| 兵器 | 静物・機械 | 弩砲／柵（オブジェクト） |

## 2. 味方28種・個体の特徴

全て「青基調＋兵種の形」。その上でグループ内の見分けを定義する。

### 占領兵（青＋白金）
| ユニット | 見分けの特徴 |
|---|---|
| クレリック | 最も簡素なフード付きローブ・両手で手持ちの十字（基本の治療役／杖は持たない） |
| プリースト | 女性・重めの法衣＋十字ロッド（十字モチーフの錫杖。攻40＝武闘派） |
| ビショップ | 司教帽（ミトラ）＋十字クロジャー（大型の十字付き錫杖。硬い・防40） |
| パラディン | 全身鎧＋十字タバード／マント＋十字柄の剣（聖騎士・最も英雄的） |

### 歩兵（青＋鋼）
| ユニット | 見分けの特徴 |
|---|---|
| ノービス | 若い・軽装・小さめの剣・盾なし（新兵感） |
| ファイター | 片手剣・盾なしの標準（主役の基準形） |
| ヴァンガード | 両手の大剣・一回り重装（移7・突撃役） |
| サムライ | 東洋甲冑＋刀（攻特化70） |
| ナイト | 重装甲＋大盾（歩兵で唯一の盾持ち＝壁）（防70・最も重いシルエット） |
| ドワーフ | 低く頑強・髭・斧／槌（山歩き） |

### 弓兵（青＋緑の自然差し色）
| ユニット | 見分けの特徴 |
|---|---|
| アーチャー | 標準の弓・革鎧・矢筒 |
| ハンター | 屈強な狩人・長弓／クロスボウ（対空特化65） |
| エルフ | 長耳・優美・長弓・森のマント（対空85） |

### 魔法兵（青紫＋発光）
| ユニット | 見分けの特徴 |

| ウィッチ | つば広帽・細身・長射程の杖（射程5） |

### 斥候（青＋暗色・軽量）
| ユニット | 見分けの特徴 |
|---|---|
| シーフ | 女性・フードは下ろす（被らない）＋ボブ／ヘアバンド・片手ダガー＋腰の巾着（盗品袋）・へそ出し軽装・ニーハイ（最速移8）。二刀はニンジャに譲る |
| ハーフリング | 小柄・スリング（射程2・撃って退く／弓ではない） |
| ニンジャ | 覆面＋暗装束・二刀 ninjato（移7・物理） |
| くノ一 | 女性の忍・くノ一装束・忍術（魔法攻撃）＝巻物のみ（刃物は持たない／魔法光・貫通0.5）・対空可（空20）。ニンジャと男女＆武器（巻物の魔法 vs 二刀の物理）で差別化（移7） |
